## Навыки, которые демонстрируют работы

- Работа с табличными данными: фильтрация, сортировка, группировка
- Формулы Excel для расчёта показателей
- Сводные таблицы и агрегация данных
- Визуализация результатов для отчётности
- Структурирование ответа: от постановки задачи к итоговым выводам

---

## Как смотреть

1. Откройте файл **ТЗ** — ознакомьтесь с постановкой задачи.
2. Откройте файл **Выполненное ТЗ** или **Тестовое задание с решением** — сравните подход к решению.
3. Обратите внимание на оформление: читаемые таблицы, понятные названия листов, логика расчётов.
